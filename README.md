dict
====

命令行下翻译工具

---
#### Install

```
git clone https://github.com/tang45/dict.git
cd dict
sudo mv ./dict.py /usr/bin/dict
sudo chmod +x /usr/bin/dict
```

#### Usage

```
dict dick
###################################
#  dick 迪克 (U: dik E: dik )
#  n. 阴茎，鸡巴；侦探；誓言
###################################
```


```
dict 鸡巴
###################################
#  鸡巴 dick (拼音: jī bā )
#  dick
###################################
```
